# MagdyPOS

هذا الكود مبني بأدوات الذكاء الاصطناعي و هو متاح لاي استخدام لوجه الله لمن يريد الاستفادة منه

لبناء الكود للاستخدام:

dotnet publish -c Release -r win-x64 --self-contained true /p:PublishSingleFile=true

للمميزات و التفاصيل برجاء زيارة الرابط التالي: 
[Link Here](https://magdypos.github.io/)
